# Key Features

## Document Management

- **Hierarchical Structure**: Organize content into spaces and nested documents.
- **Flexible Types**: Support for notes, spaces, and containers.
- **Versioning**: Integrated revision history to track changes.
- **Organization**: Favorites, recent views, and powerful search/filtering.

## Rich Text Editor

- **WYSIWYG Experience**: Built on Lexical for a fluid writing interface.
- **Markdown Support**: Full markdown shortcut support.
- **Mathematics**: Render complex equations with KaTeX.
- **Diagrams**: Create flowcharts, sequence diagrams, and more with Mermaid.
- **Code Blocks**: Syntax highlighting for code snippets.
- **Media**: Image support with cropping and optimization.

## Collaboration & User Features

- **Real-time Updates**: WebSocket integration for live data synchronization.
- **Authentication**: Secure user authentication and management.
- **Profiles**: customizable user profiles with avatars and cover images.
- **Settings**: Granular editor preferences.

## Performance

- **Optimized**: Code splitting, lazy loading, and optimized bundle sizes.
- **PWA**: Progressive Web App support for offline capabilities.
